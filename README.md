# Patient-Data-Analysis-KNIME
## Overview:
This project focuses on analyzing healthcare data using KNIME to extract meaningful insights related to patient recovery, treatment effectiveness, and follow-up compliance. Multiple datasets are integrated to provide a complete view of patient health and treatment outcomes.
## Problem Statement:
Hospitals generate large amounts of patient data, but extracting useful insights from it is challenging. This project aims to analyze patient, treatment, and follow-up data to identify patterns that can improve decision-making and patient care .
## Tools and Technologies:
KNIME Analytics Platform
Data Processing & Visualization Techniques
## Workflow
Data simulation / import using Table Creator or CSV Reader
Handling missing values (data preprocessing)
Filtering relevant patient segments (age, diagnosis, follow-up status)
Joining datasets using Patient ID
Aggregation using GroupBy (recovery rates, doctor performance)
Visualization using charts (bar chart, box plot, heatmap)
## Key Insights:
Recovery rates vary across different diagnoses
Treatment effectiveness differs between doctors
Missed follow-ups negatively impact patient recovery
Data-driven monitoring can improve healthcare outcomes
## How to Run:
Open KNIME Analytics Platform
Import the .knwf.zip workflow file
Execute all nodes
View outputs and visualizations
